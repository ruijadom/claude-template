# claude-template

Claude Code setup for React 19 / TypeScript / Vite projects — CLAUDE.md, scoped rules, skills, and scaffold command for module-based architecture.

## Stack

React 19 · TypeScript · Tailwind CSS · Vite · Vitest · Playwright · React Query · Zustand

## What's included

- **CLAUDE.md** — Project conventions, structure, and commands
- **Scaffold command** (`/scaffold <module>`) — Generates a new feature module with components, hooks, requests (React Query), store (Zustand slices), barrel exports, and test skeletons
- **Review command** (`/review`) — Code review checklist against staged changes
- **Project rules** — Accessibility and testing rules applied automatically
- **Frontend design skill** — Guidelines for distinctive, non-generic UI
